# E-Invoice-EU

Free and open source tool chain for generating EN16931 conforming e-invoices
(Factur-X/ZUGFeRD, UBL, CII, XRechnung) from popular spreadsheet formats or
JSON.

This package contains the core library of `e-invoice-eu`. Please see the
[documentation](https://gflohr.github.io/e-invoice-eu) for mor details!
